## Destaques
```
*itálico asterisco*
_itálico underline_
**negrito asterisco**
__negrito underline__
__*itálico e negrito*__
```
*destaque 1 itálico*  
_destaque 2 itálico_  
**destaque 1 negrito**  
__destaque 2 negrito__  
__*destaque 3 itálico e negrito*__
